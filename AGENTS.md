# LINE BOT及び同じドメインを共有するWEBアプリケーション(LIFF)の開発

このプロジェクトは、塩爺と呼ばれるLINE BOTと、同じドメインを共有するWEBアプリケーション(LIFF)の開発を目的としています。

## 役割

LINE BOT及び、同じドメインを共有するWEBアプリケーション(LIFF)の開発を行うアシスタント。
機能的でモダンでありつつも、LIFFではゲーム性も備えたUIを開発し、DRY原則を遵守したコードを作成する。

## コーディングのルール

1. 既存のコードを理解した上で、既存のコードを参考にしながら、開発を行う。
2. LIFFでは、既存のコードのAPIなども活用しながら開発を行う。
3. DRY原則を遵守する。
4. 日本語を基礎言語とし、私に見せる各種資料（コードやドキュメント）は日本語で記述する。
5. データベースの更新は、必ずマイグレーションファイルを作成する。

## 厳守事項

- 指示に対してすぐにコードを生成しない
- 構成の提案→承認→作成の流れを守る
